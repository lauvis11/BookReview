# BookReview

An interactive book review platform featuring an integrated AI assistant, user authentication, and comprehensive book tracking capabilities. Built for readers to discover, rate, and discuss their favorite books with the help of artificial intelligence.

## Tech Stack

**Client:** React, React Router DOM, TailwindCSS, Vite, Lucide React

**Server:** Node.js, Express, MySQL, Google Generative AI (Gemini), JSON Web Tokens (JWT), Bcrypt

## Features

- **AI Assistant:** Integrated AI to help users with book recommendations and insights (Powered by Google Generative AI).
- **User Authentication:** Secure login and registration with JWT and password hashing (Bcrypt).
- **Book Tracking:** Users can add books to favorites, rate them, and leave comments.
- **Reading Status:** Track the status of the books you are reading (e.g., Reading, Completed).
- **User Profiles:** Personalized user profiles.
- **Security & Performance:** Implemented rate limiting and Helmet for API security.
